# 短线战法 100 条（原文 + 规则映射）

> 本文件为战法**原始条文存档**。每条原文保持不变；对于已被 `stock_diagnosis()` 编码为自动判定的**可量化规则**，在条文后用 `→ 【已编码】` 标注它对应计算层的哪个判断，便于核对与维护。
>
> 核心三要素：**① 均线位置（MA5 / MA24生命线 / MA60季线）· ② 量能状态（均量线金叉死叉、持续放量）· ③ 所处高低位**。
> 计算层函数见 `../SKILL.md` 的「Layer 2 战法计算层」。

## 一、上涨特征与买入

**01、上涨之前的特征**：第一，成交量必须有所变化，股价未涨量先涨，温和放量是股价加速的有效条件；第二，股价突破并站稳在五日均线之上。
→ 【已编码】`compute_vol_ma.sustained_volume`（持续放量）+ `compute_ma.above_MA5`（站上5日线）。

**02、第一次上涨途中出现缩量涨停最安全。**
→ 【部分编码】`detect_patterns` 识别「涨停」+ `compute_vol_ma` 判缩量；"第一次"需人工结合位置判断。

**03、偏离五日均线不远时，还会出现急速拉升。**
→ 【已编码】`compute_ma.deviate_MA5_pct`（偏离5日线百分比）。

**04、五日均量线一定要处于上升通道，最好是与十日均量线形成金叉后平行向上。**
→ 【已编码】`compute_vol_ma.VMA5_x_VMA10`（金叉）+ `VMA5_slope=up`（平行向上）。买入信号项。

**05、不涨停不买进；不放量不买进；不突破不买进。经常涨停的股票一定要重点关注。**
→ 【已编码】买入三前提：`detect_patterns`涨停 + `compute_vol_ma`放量 + `compute_ma`突破5日线。

**06、当快速放量涨停的股票出现停滞或涨速跟不上时，要小心风险，果断了结。**
→ 【人工】需盘中实时涨速，盘后数据不足以判定。

**07、股价高位放量，均量线出现死叉，出货为上策。**
→ 【已编码】`compute_vol_ma.VMA5_x_VMA10=dead_cross`（弱势时计入卖出信号）。

**08、大幅暴炒过的股票短时间内不再参与。** → 【人工】

**09、三波明显上涨过后，坚决出局。** → 【人工】波段计数需主观划分。

**10、底部放量涨停，快速介入。**
→ 【部分编码】`detect_patterns`涨停 + 位置（距40日高点回撤大=底部）。

**11、中途缩量开盘涨停，大胆持股。** → 【部分编码】涨停 + 缩量。

**12、高位放量收阴，偏离五日均线太远，果断了结。**
→ 【已编码】`detect_patterns`「高位放量收阴」+ `compute_ma.deviate_MA5_pct`。卖出信号项。

**13、高位量能萎缩，注意调整出现，快速出货，观望。** → 【部分编码】量能萎缩 + 高位。

**14、不要孤立看待一个股票的走势，注意观察板块联动效应。**
→ 【数据支持】`industry_comparison()` 行业排名 + `ths_hot_reason()` 题材。

**15、跳空越大，涨停越快，主力越强。** → 【已编码】`detect_patterns`「向上跳空缺口」。

**16、上涨途中的激烈震荡是不可避免的，关键要看结果。** → 【人工】

**17、流通盘5000万以下的小盘股涨停，最好保持7%的换手率。**
→ 【数据支持】`eastmoney_stock_info` 取流通股 + 腾讯换手率。

**18、二十四日均线可称为生命线。股价运行在生命线之上，意味该股有生命力；在此线之下，即是失去活力的股票，短线暂时可不关注。**
→ 【已编码·总纲】`compute_ma.above_MA24`。`stock_diagnosis` 以此为多空总分界。

**19、在二十四日均线之下，三日之内不能重上该线上方，则判断为主力出货，后市不看好。**
→ 【部分编码】`above_MA24=False` + 近3日未收复。

**20、股价沿着五日均线上升的时候开始留意。小阳小阴的组合K线是加速之前的征兆。** → 【人工】

**21、向上跳空缺口是强势股的最常见特征，不回补更强。**
→ 【已编码】`detect_patterns`「向上跳空缺口」。

**22、光头大阳线可能是股价起飞的开始，成交量必须放大。**
→ 【已编码】`detect_patterns`「光头大阳」+ 量能放大。

**23、如要获利，应有临场作战时热点联想与板块联想的习惯。当一只股涨停时，要第一时间联想到什么股票的板块、图形和题材等与之相同。**
→ 【数据支持】`baidu_concept_blocks()` 概念归属 + `ths_hot_reason()` 题材归因。

**24、缩量涨停预示空间更大，主力资金更充足。** → 【部分编码】涨停 + 缩量。

**25、能不断创出新高就是主力的货没有出完，还要涨。** → 【部分编码】创新高检测。

**26、强势股不破五日均线则不考虑出货。** → 【已编码】`compute_ma.above_MA5`。

**27、如是追高买进的，通常做法是在次日开盘高开3%就走。** → 【人工·交易纪律】

**28、不要在高位追进涨停股，最好是寻找底部潜力股。** → 【人工】

**29、经过震仓的黑马更加可靠。第三天缩量突破震仓价，短线跟进。** → 【人工】

**30、均量线平行向上是成交量最理想的状态。**
→ 【已编码】`compute_vol_ma.VMA5_slope=up` + `VMA5_above_VMA10`。

**31、偏离均线越远，风险越大。** → 【已编码】`compute_ma.deviate_MA5_pct`。

**32、不管何种黑马形态，遇到高位五日线死叉生命线，是极为可靠的卖出信号。**
→ 【已编码·关键】`compute_ma.MA5_x_MA24=dead_cross`（且未收复生命线才计入）。

**33、投资正道：先分析大盘，选好个股，然后具体操作策略。**
→ 【已编码】`market_regime()` 大盘多空环境。

**34、短线时间：两周以内，三天以上。超短线时间：三天以内。** → 【人工·时间纪律】

**35、"神*手买入法"操作细则：**
- 五日均线上穿二十四日均线（MACD平行向上），可作为买入信号。准确性一般。
- 上升通道股，在第一次回调到二十四日均线受到支撑买进。比较准确。
- 上升通道股，在第二次回调到二十四日均线受到支撑买进。非常准确。
- 上升通道股，在第三次回调到二十四日均线受到支撑买进。很准确。
- 经过三波上涨、加速拉升，股价在高位，五日均线下穿二十四日均线，股价进入死亡区，为空仓信号。应注意风险，回避为主。
→ 【已编码】MA5上穿/下穿MA24（`MA5_x_MA24`）+ `compute_indicators.MACD_above_signal`；回调次数需人工。

**36、趋势确立，买进后多看少动。** → 【人工·持有纪律】

**37、持续放量是上涨的保证，量萎缩则很难涨。** → 【已编码】`compute_vol_ma.sustained_volume`。

**38、大盘股底部一次建仓，高位分批卖出。** → 【人工】

**39、高位五日均线拐头向下，坚决退出观望。** → 【已编码】`compute_ma.MA5_slope=down`（高位时）。

**40、强势股通常盘整三天，在第四天启动，或者是八天。盘整之后启动的当天短线介入比较安全。这就是"下跌衰竭"切入点。** → 【人工】

**41、若在高位出现连续"十字星"，应注意顶部来临。**
→ 【已编码】`detect_patterns`「十字星」（高位预警）。

**42、对于中线确认上涨的股票止跌后，第一根阳线时买进。** → 【人工】

**43、急速拉高的时候，可在缩量之后卖出，在回抽时捡回筹码。** → 【人工】

**44、两波主升浪上涨要减磅，三波大涨不再创新高出货。** → 【人工·波段】

**45、大盘股从最高价回调到最低价，一般为三、五、七天。其中五天比较多，在第五天涨稳收阳时可介入。** → 【人工】

**46、上涨与温和成交量配合才比较安全。量均线平行向上，容易出现加速。**
→ 【已编码】`above_MA5` + `sustained_volume` + `VMA5_slope=up`。

**47、寻找突破的个股作为短线跟踪的对象。突破重要阻力位进入上升轨道的股票有快速获利可能。** → 【人工】

**48、日、周均线都为多头，则易出黑马。**
→ 【已编码】`compute_ma.bull_align`（日）+ `weekly_view.weekly_MA5_slope=up`（周）。

**49、缩量突破大量震仓位，是最好的买入点。** → 【人工】

**50、第一个缩量涨停是最安全的。** → 【部分编码】涨停 + 缩量 + "第一个"需人工。

**51、五日均线是短线强弱的分界线。跳空突破不回补是强势的特征。**
→ 【已编码】`above_MA5` + `detect_patterns`「向上跳空缺口」。

**52、理想的短线涨停股：封单果断，足够大；抛压很小。封停超过十五分钟不打开就比较安全。**
→ 【数据支持】`limit_up_quality()` 封单量；封停时长需逐笔（盘中）。

**53、突破盘整区买进，可安全获利。** → 【人工】

**54、前一交易日收阴线，第二天出现涨停，比较安全。**
→ 【部分编码】昨阴 + 今涨停（`detect_patterns`）。

**55、要注意中午抢盘的股票下午是否有短线机会。** → 【人工·盘中】

**56、多关注盘升将要加速的股，且在震仓过后及时跟进。** → 【人工】

**57、缩量创新高是比较安全的买入信号。** → 【部分编码】缩量 + 创新高。

**58、突破箱顶可能出现加速的机会。** → 【人工】

**59、牛股的第一次回调是介入的良好时机。** → 【人工】

**60、慢步盘升的股票最容易出现加速，而暴利通常都是从加速上涨的股票上获得。** → 【人工】

**61、强势上涨特征：连续收四根阳线，调整一两天，又出现三根或四根阳线。不断大涨小回就是好股票的特征。**
→ 【已编码】`detect_patterns`「红三兵」（连续阳线递增）。

**62、在快速放量加速阶段，成交量一旦跟不上，就是短线出货最佳时机。若同时出现：均线空头刚死叉；五日均量线死叉十五日均量线；大盘空头向下趋势——应果断出货。**
→ 【已编码】`compute_vol_ma`死叉 + `market_regime=bear` + `compute_ma`空头。

**63、如果成交量进入比以前放大几倍且能够持续的状态，股价可能进入拉升阶段。**
→ 【已编码】`compute_vol_ma.vol_ratio_3d` + `sustained_volume`。

**64、注意成交量的变化，有机构关照的股票一定是成交活跃的。成交量温和持续放大是资金介入的重要特点。**
→ 【数据支持】`compute_vol_ma` + `stock_fund_flow_120d()` 主力净流入。

**65、高位涨停之后大幅震荡时，最好离场观望。** → 【人工】

**66、五日均线和生命线平行向上是比较明确的上升通道，注意把握回档或横盘几天后介入。**
→ 【已编码】`MA5_slope=up` + `MA24_slope=up` + `MA5_above_MA24`。

**67、盘中二次放量和日线放量是分析股价推动的重要依据。** → 【部分编码·盘中】

**68、上涨斜率越大，机会越大，同时风险也越大。** → 【人工】

**69、以30度斜率上涨的特征，经常会在相同的股票上重现。** → 【人工】

**70、周线出现沿五日均线上涨，最后为连续带下影阳线的股票，容易催生大阳。**
→ 【已编码】`weekly_view` + `detect_patterns`「长下影线」。

**71、周线出现三线合一的信号，注意是否中线机会到来。**
→ 【已编码】`weekly_view.three_line_converge`。

**72、均量线金叉、均线金叉、股价突破等多重共震时，则机会更大。**
→ 【已编码】多信号叠加：`compute_vol_ma`金叉 + `compute_ma`金叉 + 突破。

## 二、行业与新股

**73、有色金属行业的股价有时会受到商品期货走势的影响。** → 【人工】

**74、通常第二波上涨是最猛、最强的上升浪。** → 【人工·波段】

**75、股价远离生命线，如均线不能跟上，容易出现回档。**
→ 【已编码】`deviate` 大 + `MA24_slope`。

**76、高位巨量应警惕机构对敲出货，若在底部则为启动信号。**
→ 【部分编码】巨量 + 高低位判断 + `stock_fund_flow_120d` 验证。

**77、以涨停突破下降通道的次新股是短线最强的股票。** → 【人工·次新】

**78、重点关注上市后出现第一个涨停的新股。中线关注刚上市就出现涨停的新股。**
→ 【数据支持】`eastmoney_stock_info` 上市日期 + 涨停。

**79、大幅波动、大幅上涨是新股暴涨的重要特征。** → 【人工】

**80、创历史新高的新股，每日换手率在7%以上为好。** → 【数据支持】换手率 + 创新高。

**81、经常关注创新高的股票，你将成为获利最快的投资者。** → 【部分编码】创新高检测。

## 三、洗盘、卖出与风控

**82、启动之前都要经过巨幅的洗盘震仓。股价启动前和在高位时，都会出现巨幅的波动。** → 【人工】

**83、大涨小回是主升段的最主要特征。** → 【人工】

**84、连续三天放量才是大黑马启动的重要标记。只放量一天，只是短暂的行为。**
→ 【已编码】`compute_vol_ma.sustained_volume`（近3日量均放大）。

**85、放量突破平台，是行情启动的信号。** → 【部分编码】放量 + 突破。

**86、不断创新高的强势股：以五日均线值为买入价格，如跌破五日均线后三天不能重新站上该线止损或止盈。涨幅过大（超过50%）的不介入为好。**
→ 【已编码】`above_MA5` + 涨幅过滤。

**87、高位大幅震荡或放出巨量、五日均线从加速向上到走平时，应注意风险，以防主力出货。**
→ 【已编码】`MA5_slope` 由 up 转 flat（高位）。

**88、高位大阴线，第一时间出局；高位跌破五日均线，第一时间减少仓位；高位速破生命线，第一时间清仓；跌破季线，空仓休息。**
→ 【已编码·关键】`above_MA24`（生命线）+ `above_MA60`（季线）。卖出信号项。

**89、毛毛虫K线是庄股的特征。股价在高位，且无成交量，绝对不碰。** → 【人工】

**90、均线反压且向下发散，是股价即将加速下跌的信号。**
→ 【已编码】`MA5_slope=down` 且 `MA24_slope=down`（空头向下发散）。卖出信号项。

**91、当股价盘跌，反弹生命线无力再上涨时，应清仓。**
→ 【部分编码】`above_MA24=False` + 反弹乏力。

**92、对弱势股坚决不等反弹；不做反弹行情；不随便抄底。**
→ 【已编码·纪律】生命线下方一律「回避」，不因超卖反弹给买点。

**93、对于不明原因的暴跌个股，不要轻易做第一次反弹。** → 【人工】

**94、对于老庄股，不要轻易在股价高位冒险追高。** → 【人工】

**95、发现机会时，缩小K线图查看总体股价及目前所处位置；再看周线图确认。决定买入时，依次查看60分钟、30分钟、15分钟图以确定买入点（莫忘看一眼F10个股基本资料）。**
→ 【已编码·流程纲领】对应看盘九步：位置→周线（`weekly_view`）→分钟（mootdx分钟K）→F10（`mootdx F10`）。

**96、在行情比较好时，可操作涨势凌厉的红三兵。一般红三兵后，第四天小阴线，第五天放量大涨。**
→ 【已编码】`detect_patterns`「红三兵」。

**97、在大盘暴跌时，低价底部逆市逞强的个股就是下一阶段重点关注的对象。**
→ 【数据支持】`market_regime=bear` + `ths_hot_reason` 逆势强势股。

**98、主要研究对象：成交量的变化；K线运行规律；均线形态特征。**
→ 【已编码·总纲】对应三大计算模块：`compute_vol_ma` / `detect_patterns` / `compute_ma`。

**99、威廉指标小于50时为强势。可根据该指标长期在强势区域进行操作，以提高成功机率。**
→ 【已编码】`compute_indicators.WR_strong`（WR6 < 50）。

**100、连续三天以上放量上涨，是股价发生重大变化的起点。这是技巧。**
→ 【已编码】`compute_vol_ma.sustained_volume`（连续放量）+ 上涨。
