## ② 大盘环境（规则33/62/97，来自 TradingAgents 市场分析师）

先看大盘再选股。上证/深成/创业板 指数 MA5/10/20 多空排列 → bull / neutral / bear。
- 大盘空头时配合个股出货（规则62）；大盘暴跌时找逆势逞强股（规则97）。
- 函数：`market_regime`

## ③ 游资 / 连板语义（来自 TradingAgents 游资追踪分析师）

| 判据 | 阈值/语义 |
|------|----------|
| 量价异动 | 日量 > 20日均量 **2倍** = 放量异动；换手率 >10% = 异常活跃 |
| 连板语义 | 首板**放量=分歧** / **缩量=一致**（缩量更安全，规则24/50）；**三板以上=妖股模式**需特别谨慎（规则09/76） |

函数：`volume_anomaly`。妖股模式会让 `risk_check` 直接标风险、`stock_diagnosis` 降级。

## ④ 资金博弈四态（来自 TradingAgents 游资框架）

综合**主力净 + 散户净(中小单) + 当日涨跌**判定四态：

| 态 | 条件 | 含义 |
|----|------|------|
| 主力吸筹 | 主力净流入 + 股价未涨 | 低位吸筹 |
| 主力推动 | 主力净流入 + 上涨 + 超大单>0 | 机构推动 |
| 游资接力 | 主力净流入 + 上涨 + 超大单不足 | 大单为主、游资特征 |
| 主力出货 | 主力净流出 + 下跌 | 出货 |
| 散户主导(诱多) | 主力净流出 + 上涨 | 散户接盘/拉高出货嫌疑 |

函数：`capital_battle`（数据 `fund_flow_em`，走 push2，间歇可达，拿不到标 `[数据缺失:资金流]`）。

## ⑤ 基本面（来自 TradingAgents 基本面分析师）

| 判据 | 阈值 |
|------|------|
| 亏损 | 净利润为负 → 爆炒亏损股风险 |
| ROE | 净利润/净资产 |
| 资产负债率 | (流动+长期负债)/总资产 **>70%** 预警 |
| 现金流质量 | 经营现金流/净利润 **<0.3** = 利润含金量低（纸面利润） |
| 估值参照系 | **A股PE中位数30-50x为常态**，对标同行业横向比，**勿套用美股15-25x**；PE>200即便A股也偏贵 |

函数：`risk_check`（财务取自 mootdx finance，估值取自腾讯）。

## ⑥ 解禁压力（来自 TradingAgents 解禁监控分析师）

| 评级 | 阈值（解禁占流通比） |
|------|---------------------|
| 重大压力 | >20% |
| 中等压力 | 5~20% |
| 轻微压力 | <5% |
| 无压力 | 未来90天无待解禁 |

减持新规提示：大股东每90天集中竞价≤总股本1%、大宗≤2%；股价低于解禁成本则减持概率大降。
函数：`lockup_pressure`（数据 `lockup_expiry`，东财 datacenter）。

## ⑦ 人工纪律提示（战法100条里约30条经验规则）

工具算不了、但战法很看重的经验铁律，按形态特征主动提醒：
- 规则93：暴跌后第一次反弹不轻易做（近期跌停+今日反弹时触发）
- 规则27/28：涨停且偏离5日线过远，不追高
- 规则09/44：接近高位仍上行，留意第几波，三波后出局
- 规则41：高位十字星，顶部预警
- 规则82/83：剧烈波动，辨洗盘还是出货
- 规则33/36/34：买前看大盘、买后多看少动、短线持有2周内
- 规则92：弱势区反弹不抄底

函数：`manual_reminders`。

## ⑧ 多空对辩（来自 TradingAgents 多头/空头研究员框架）

把已算信号归类为三方观点，由 Claude 据此对辩式综合（形似多 Agent 辩论，实为一个分析师的全面双向思考）：

- **🐂 多头席**（多头框架）：站稳生命线 + 买入信号 + 资金面吸筹/接力 + 连板良性 + MACD多头
- **🐻 空头席**（空头框架）：卖出信号 + 风险提示 + 资金面出货/诱多 + 妖股 + **T+1陷阱**（今日涨停追高、明日才能卖、隔夜反转风险，规则27/28）
- **⚖️ 战法裁决**：以生命线为总纲，权衡多空条数 → 五档评级

函数：`bull_bear_debate`。

## 五档评级判定逻辑（stock_diagnosis）

| 评级 | 条件 |
|------|------|
| **Buy** | 涨停启动 + 买入信号≥4 + 无风险（亏损股不给） |
| **增持** | 买入信号≥3 + 无卖出（亏损股封顶为持有） |
| **持有** | 站上生命线但有卖出信号 / 信号中性 |
| **减持** | 生命线下方信号不足 / 妖股顶部降级 |
| **卖出** | 生命线下方 + 卖出信号≥2 |

> ⚠️ 全部为离线纯规则的「技术形态 + 基本面避雷」体检，**不构成投资建议**。
